# V-STATION 프로젝트 규칙

## 프로젝트 개요
- **타입**: 심리테스트/바이럴 콘텐츠 플랫폼
- **디자인 스타일**: Playful (Global 규칙의 Style 2)
- **타겟**: MZ세대, 바이럴 공유 목적

---

## 디자인 원칙

### 핵심 컨셉: "놀이터"
- 딱 봐도 "재미있는 테스트 사이트"임이 느껴져야 함
- 방구석연구소, 쿠키런 같은 친근하고 개성있는 분위기
- **AI가 만든 티 절대 금지** - 뻔한 카드 나열 X

### 테마별 분리 (필수)

| 테마 | 스타일 | 분위기 |
|------|--------|--------|
| Identity | 잡지/에디토리얼 | Serif 폰트, 흑백 기조, 고급스러움, "나를 탐구" |
| Retro | Windows 95 | 복고 UI, 회색톤, 픽셀 요소, 노스탤지어 |

**중요**: Identity와 Retro는 완전히 다른 UI로 구현해야 함

---

## Identity 섹션 디자인

### 스타일: 에디토리얼/잡지
- 잡지 표지, 기사 목록 느낌
- 세련되고 지적인 분위기

### 요소
```
- 폰트: Georgia, serif (제목), system-ui (본문)
- 색상: 흑백 기조 + 단색 포인트
- 레이아웃: 비대칭 그리드, 기사 번호, 카테고리 태그
- 호버: 밑줄 애니메이션, 미니멀한 효과
```

### ❌ 금지
- 파스텔 배경
- 둥근 카드
- 귀여운 아이콘

---

## Retro 섹션 디자인

### 스타일: Windows 95 바탕화면
- 진짜 복고풍 OS 느낌
- 노스탤지어, 레트로 게임

### 요소
```
- 배경: #008080 (Windows 청록색)
- UI: 회색 창 (#c0c0c0), 파란색 타이틀바
- 레이아웃: 바탕화면 아이콘 흩어진 배치
- 테두리: 입체 효과 (border 색상 조합)
- 파일명: xxx.exe 형식
```

### 컴포넌트
- `RetroWindow`: Windows 창 스타일
- `RetroButton`: 입체 버튼
- `RetroProgress`: 줄무늬 프로그레스바

### 참고
- Windows 95/98 UI
- MS-DOS
- 레트로 게임기

---

## 전체 공통 요소

### 스티커/장식
- 떠다니는 배지 (NEW, HOT, MZ pick)
- 손그림 요소
- 불규칙하게 배치된 장식

### 애니메이션 (Framer Motion)
```typescript
// 흔들림
animate={{ rotate: [0, -5, 5, 0] }}

// 떠다니기
animate={{ y: [0, -10, 0] }}

// 깜빡임
animate={{ scale: [1, 1.1, 1] }}
```

### 텍스처
- 노이즈 오버레이 (종이 질감)
- 격자 패턴 배경

---

## 컬러 시스템

### 전체 배경
```
베이스: amber-50 (따뜻한 크림색)
```

### Identity 팔레트
```
배경: white
텍스트: black / gray-500
포인트: black
```

### Retro 팔레트
```
배경: #008080 (청록)
창: #c0c0c0 (회색)
타이틀바: #000080 → #1084d0 (그라데이션)
테두리: #dfdfdf (밝음) / #404040 (어두움)
```

## 금지 사항 (이 프로젝트 특화)

### 레이아웃
- 동일 크기 카드 그리드
- 수평 스크롤 카드 리스트
- 모든 카드 같은 디자인

- Identity에 파스텔 색상
- Retro에 현대적 UI
- 테마 구분 없이 섞어 쓰기

### 기타
- rounded-2xl + shadow 조합
- "Welcome!", "Let's go!" 같은 AI 전형 문구
